# ⚽ 網頁版足球遊戲

Python Flask + HTML5 Canvas 單機伺服器足球遊戲。

## 專案結構

```
soccer_game/
├── app.py                  # Flask 後端（提供靜態網頁）
├── templates/
│   └── index.html          # 遊戲前端（HTML5 Canvas + JavaScript）
└── README.md               # 本說明文件
```

## 安裝與啟動

### 1. 安裝 Flask

```bash
pip install flask
```

### 2. 啟動伺服器

```bash
python app.py
```

> 若系統有多個 Python 環境（如 Anaconda），請指定完整路徑：
> ```bash
> C:\Users\<你的帳號>\AppData\Local\anaconda3\python.exe app.py
> ```

### 3. 開啟遊戲

用瀏覽器開啟：**http://127.0.0.1:5000**

## 操作說明

| 按鍵 | 功能 |
|------|------|
| ⬆ ⬇ ⬅ ➡ 方向鍵 | 移動玩家（藍色圓圈） |
| **快速雙擊**同方向鍵 | 衝刺加速（持續約 1.2 秒，出現黃色光暈） |
| **Space 空白鍵** | 射門（帶球時有效，朝目前朝向射出） |
| **ESC** | 暫停 / 繼續遊戲 |
| **Q** | 從暫停畫面返回主選單（比分歸零） |

## 難度選擇

畫面下方有三個難度按鈕，**遊玩中也可即時切換**：

| 難度 | 電腦速度 | 電腦衝刺速度 | 衝刺頻率 |
|------|---------|------------|---------|
| 簡單（綠） | 慢 | 慢 | 很少 |
| 普通（橘） | 中 | 中 | 偶爾 |
| 困難（紅） | 快 | 快 | 頻繁 |

## 遊戲規則

- **搶球**：走向足球，碰到即可帶球（球員出現黃色邊框）
- **帶球**：球固定跟在球員朝向的前端
- **射門**：帶球後按 **Space** 往目前朝向射出
- **搶斷**：衝向帶球對手並碰到即可奪球（雙方皆適用）
- **電腦 AI**：自動追球、帶球往玩家球門移動、靠近球門自動射門、定期衝刺
- **計分**：球進對方球門得 1 分，進球後雙方回到初始位置，球重新生成
- **暫停**：按 **ESC** 暫停，再按一次繼續；按 **Q** 可返回主選單
