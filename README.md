# ⚽ 網頁版足球遊戲

Python Flask + HTML5 Canvas 單機伺服器足球遊戲，支援電腦鍵盤與手機觸控操作。

Deployed by **Kyle Cheng**

## 專案結構

```
soccer_game/
├── app.py                  # Flask 後端（提供靜態網頁）
├── templates/
│   └── index.html          # 遊戲前端（HTML5 Canvas + JavaScript）
├── requirements.txt        # Python 套件清單（flask、gunicorn）
├── Procfile                # Railway / Heroku 啟動指令
└── README.md               # 本說明文件
```

## 安裝與啟動（本機）

### 1. 安裝 Flask

```bash
pip install flask
```

### 2. 啟動伺服器

```bash
python app.py
```

> 若系統有多個 Python 環境（如 Anaconda），請指定完整路徑：
> ```bash
> C:\Users\<你的帳號>\AppData\Local\anaconda3\python.exe app.py
> ```

### 3. 開啟遊戲

用瀏覽器開啟：**http://127.0.0.1:5000**

## 操作說明

### 電腦鍵盤

| 按鍵 | 功能 |
|------|------|
| ⬆ ⬇ ⬅ ➡ 方向鍵 | 移動玩家 |
| **持壓**同方向鍵 | 漸進加速（約 2 秒升至 200%），HUD 顯示即時速度百分比 |
| **快速雙擊**同方向鍵 | 立即衝刺至最高速（200%） |
| **Space 空白鍵** | 射門（帶球時有效，朝目前朝向射出） |
| **ESC** | 暫停 / 繼續遊戲 |
| **Q** | 從暫停畫面返回主選單（比分歸零） |

### 手機觸控

| 操作 | 功能 |
|------|------|
| 點擊畫面 | 開始遊戲 |
| **右手**按畫面右下角圓形方向鍵 | 控制球員移動（支援斜向）；**持壓**漸進加速，最高 200% |
| **左手**點擊畫面左下角紅色「射門」按鈕 | 射門（帶球時有效） |
| 點擊畫面左上角「←」按鈕 | 返回主選單（比分歸零） |
| 點擊畫面下方「⏸ 暫停」按鈕 | 暫停 / 繼續遊戲 |
| 右上角「⛶」按鈕 | 切換全螢幕（橫屏時自動嘗試，Android 支援；iOS 需手動點按） |

> 電腦與手機操作可同時使用，互不干擾。觸控按鈕均繪製於 Canvas 角落，不遮擋遊戲人物。

## 難度選擇

畫面下方有三個難度按鈕，**遊玩中也可即時切換**：

| 難度 | 電腦速度 | 電腦衝刺速度 | 衝刺頻率 |
|------|---------|------------|---------|
| 簡單（綠） | 慢 | 慢 | 很少 |
| 普通（橘） | 中 | 中 | 偶爾 |
| 困難（紅） | 快 | 快 | 頻繁 |
| 地獄（橙紅） | 極快（超過玩家基礎速度） | 極快（超過玩家衝刺極限） | 極頻繁，幾乎全場射門 |

## 遊戲規則

- **球員外觀**：人形角色（頭、三支尖髮、球衣、球褲、雙腳）；移動時雙腳動態擺動；玩家為藍色配深棕髮，電腦為紅色配黑紫髮
- **搶球**：走向足球，碰到即可帶球（球衣出現黃色邊框）
- **帶球**：球固定跟在球員朝向的前端
- **射門**：帶球後按 Space（或點左下角按鈕）往目前朝向射出
- **搶斷**：衝向帶球對手並持續接觸 **1 秒**即可奪球（電腦搶玩家需蓄力，玩家搶電腦即時成功）
- **電腦 AI**：自動追球、帶球往玩家球門移動、靠近球門自動射門、定期衝刺
- **計分**：球進對方球門得 1 分，進球後雙方回到初始位置，球重新生成
- **遊戲時限**：每局 **60 秒**，倒數計時顯示於比分板左側；時間到自動顯示結算畫面（🏆 獎盃 + 勝方公告 + 最終比分）
- **暫停**：按 ESC 或點擊「⏸ 暫停」按鈕暫停，再次操作繼續；按 Q 或手機按左上角「←」可返回主選單

## 部署

### Railway（推薦）

```bash
# 安裝 Railway CLI
npm install -g @railway/cli

# 登入（使用 GitHub 帳號）
railway login

# 部署
railway up
```

或直接在 [railway.app](https://railway.app) 連結 GitHub repo，每次 `git push` 自動重新部署。

### 本機對外分享（ngrok）

```bash
# 終端機 1：啟動 Flask
python app.py

# 終端機 2：建立公開隧道
ngrok http 5000
```

將 ngrok 產生的 `https://xxx.ngrok-free.app` 網址傳給對方即可。
