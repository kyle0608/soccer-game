# ⚽ 網頁版足球遊戲

Python Flask + HTML5 Canvas 單機伺服器足球遊戲，支援電腦鍵盤與手機觸控操作。

Deployed by **Kyle Cheng**

## 專案結構

```
soccer_game/
├── app.py                  # Flask 後端（提供靜態網頁）
├── templates/
│   └── index.html          # 遊戲前端（HTML5 Canvas + JavaScript）
├── requirements.txt        # Python 套件清單（flask、gunicorn）
├── Procfile                # Railway / Heroku 啟動指令
└── README.md               # 本說明文件
```

## 安裝與啟動（本機）

### 1. 安裝 Flask

```bash
pip install flask
```

### 2. 啟動伺服器

```bash
python app.py
```

> 若系統有多個 Python 環境（如 Anaconda），請指定完整路徑：
> ```bash
> C:\Users\<你的帳號>\AppData\Local\anaconda3\python.exe app.py
> ```

### 3. 開啟遊戲

用瀏覽器開啟：**http://127.0.0.1:5000**

## 操作說明

### 電腦鍵盤

| 按鍵 | 功能 |
|------|------|
| ⬆ ⬇ ⬅ ➡ 方向鍵 | 移動玩家（藍色圓圈） |
| **快速雙擊**同方向鍵 | 衝刺加速（持續約 1.2 秒，出現黃色光暈） |
| **Space 空白鍵** | 射門（帶球時有效，朝目前朝向射出） |
| **ESC** | 暫停 / 繼續遊戲 |
| **Q** | 從暫停畫面返回主選單（比分歸零） |

### 手機觸控

| 操作 | 功能 |
|------|------|
| 點擊畫面 | 開始遊戲 / 從暫停繼續 |
| **右手**按住畫面任意位置並滑動 | 浮動搖桿出現，控制球員移動方向 |
| 搖桿推到**邊緣 85% 以上** | 自動衝刺（搖桿變金黃色） |
| **左手**點擊左下角紅色「射門」按鈕 | 射門（帶球時有效） |

> 電腦與手機操作可同時使用，互不干擾。

## 難度選擇

畫面下方有三個難度按鈕，**遊玩中也可即時切換**：

| 難度 | 電腦速度 | 電腦衝刺速度 | 衝刺頻率 |
|------|---------|------------|---------|
| 簡單（綠） | 慢 | 慢 | 很少 |
| 普通（橘） | 中 | 中 | 偶爾 |
| 困難（紅） | 快 | 快 | 頻繁 |

|------|------|

## 遊戲規則

- **搶球**：走向足球，碰到即可帶球（球員出現黃色邊框）
- **帶球**：球固定跟在球員朝向的前端
- **射門**：帶球後按 Space（或點左下角按鈕）往目前朝向射出
- **搶斷**：衝向帶球對手並持續接觸 **1 秒**即可奪球（電腦搶玩家需蓄力，玩家搶電腦即時成功）
- **電腦 AI**：自動追球、帶球往玩家球門移動、靠近球門自動射門、定期衝刺
- **計分**：球進對方球門得 1 分，進球後雙方回到初始位置，球重新生成
- **暫停**：按 ESC 暫停，再按一次繼續；按 Q 可返回主選單

## 部署

### Railway（推薦）

```bash
# 安裝 Railway CLI
npm install -g @railway/cli

# 登入（使用 GitHub 帳號）
railway login

# 部署
railway up
```

或直接在 [railway.app](https://railway.app) 連結 GitHub repo，每次 `git push` 自動重新部署。

### 本機對外分享（ngrok）

```bash
# 終端機 1：啟動 Flask
python app.py

# 終端機 2：建立公開隧道
ngrok http 5000
```

將 ngrok 產生的 `https://xxx.ngrok-free.app` 網址傳給對方即可。
