# ⚽ 網頁版足球遊戲

Python Flask + HTML5 Canvas 單機伺服器足球遊戲，支援電腦鍵盤與手機觸控操作。

Deployed by **Kyle Cheng**

## 專案結構

```
soccer_game/
├── app.py                  # Flask 後端（提供靜態網頁）
├── templates/
│   └── index.html          # 遊戲前端（HTML5 Canvas + JavaScript）
├── requirements.txt        # Python 套件清單（flask、gunicorn）
├── Procfile                # Railway / Heroku 啟動指令
└── README.md               # 本說明文件
```

## 安裝與啟動（本機）

### 1. 安裝 Flask

```bash
pip install flask
```

### 2. 啟動伺服器

```bash
python app.py
```

> 若系統有多個 Python 環境（如 Anaconda），請指定完整路徑：
> ```bash
> C:\Users\<你的帳號>\AppData\Local\anaconda3\python.exe app.py
> ```

### 3. 開啟遊戲

用瀏覽器開啟：**http://127.0.0.1:5000**

## 操作說明

### 電腦鍵盤

| 按鍵 | 功能 |
|------|------|
| ⬆ ⬇ ⬅ ➡ 方向鍵 | 移動玩家 |
| **持壓**同方向鍵 | 漸進加速（約 2 秒升至 200%） |
| **快速雙擊**同方向鍵 | 立即衝刺至最高速（200%） |
| **Space**（帶球時） | 射門，球自動朝電腦球門飛出；距離愈遠偏差愈大 |
| **Space**（不帶球，靠近電腦時） | 搶斷衝刺：急速衝向電腦嘗試斷球 |
| **ESC** | 暫停 / 繼續遊戲（有哨音） |
| **Q** | 從暫停畫面返回主選單（比分歸零） |

### 手機觸控

| 操作 | 功能 |
|------|------|
| 點擊畫面 | 開始遊戲 |
| **右手**按畫面右下角圓形方向鍵 | 控制球員移動（支援斜向）；持壓漸進加速，最高 200% |
| **左手**點擊左下角紅色「射門」按鈕（帶球時） | 射門 |
| **左手**點擊左下角紅色「射門」按鈕（不帶球，靠近電腦時） | 搶斷衝刺 |
| 點擊畫面左上角「←」按鈕 | 返回主選單（比分歸零） |
| 點擊畫面下方「⏸ 暫停」按鈕 | 暫停 / 繼續遊戲 |
| 右上角「⛶」按鈕 | 切換全螢幕（Android 支援；iOS 不支援隱藏此鈕） |

> 橫屏時，射門鍵移至畫面最左側、方向鍵移至最右側，避免遮擋球場。

## 難度選擇

畫面下方有四個難度按鈕，**遊玩中也可即時切換**：

| 難度 | 電腦速度 | 電腦衝刺速度 | 衝刺頻率 |
|------|---------|------------|---------|
| 簡單（綠） | 慢 | 慢 | 很少 |
| 普通（橘） | 中 | 中 | 偶爾 |
| 困難（紅） | 快 | 快 | 頻繁 |
| 地獄（橙紅） | 極快（超過玩家基礎速度） | 極快（超過玩家衝刺極限） | 極頻繁，幾乎全場射門 |

## 音效

| 時機 | 音效 |
|------|------|
| 開賽 / 暫停 / 中場 / 結束 | 長哨音（高亮，約 1.2 秒） |
| 任一方射門 | 短促踢球聲 |
| 任一方得分 | 上升歡慶四音（C→E→G→高C） |
| 中場休息 | 哨音後接 6 音輕鬆旋律 |

音效由 Web Audio API 即時合成，無需外部音效檔。

## 遊戲規則

### 一般對戰

- **球員外觀**：人形角色（頭、三支尖刺髮、球衣、球褲、雙腳）；玩家藍色配深棕髮，電腦紅色配黑紫髮
- **搶球**：走向足球，碰到即可帶球（球衣出現黃色邊框）
- **帶球**：球固定跟在球員朝向的前端
- **射門**：帶球後按 Space（或射門鍵），球自動朝電腦球門方向射出。距離愈近偏差愈小，從中線後方射出可保證球到達球門
- **射球速度**：出腳瞬間最快，之後逐漸減慢（指數衰減）；過中線射門速度足夠抵達球門
- **搶斷衝刺**：不帶球時按 Space / 射門鍵，若電腦在 140px 內，玩家會急速衝向電腦；衝刺途中若觸碰到持球電腦則立即奪球（衝刺時有紅色光圈提示）
- **電腦搶球**：電腦持續接觸玩家 **1 秒**才能奪球（外圈進度條顯示蓄力）；玩家搶電腦則即時成功
- **電腦 AI**：自動追球、帶球往玩家球門推進、靠近球門自動射門、定期加速衝刺
- **計分**：球進對方球門得 1 分，進球後雙方回到初始位置，球重新生成

### 時間制度

- 每局分為**上半場**與**下半場**，各 30 秒（全場 60 秒）
- HUD 左側顯示當前半場倒數計時（上半場 / 下半場）
- 上半場結束：自動中場休息，顯示當前比分與輕鬆音效，按任意鍵 / 點畫面開始下半場
- 遊戲結束時顯示結算畫面（🏆 勝方公告 + 最終比分）

### 12碼 PK 大戰

若全場結束後**雙方平手**，自動進入 PK 大戰：

- 共 5 輪，雙方輪流各踢一球
- **玩家踢球**：瞄準指示點在球門上下振盪，按 Space / 射門鍵確認方向後射出；電腦守門員會做出判斷（65% 猜對方向）
- **電腦踢球**：電腦 1.5 秒後自動射門，玩家用 ↑↓ 方向鍵（或手機方向鍵）控制守門員上下移動攔截
- 5 輪後得分多者勝；仍平手則為**和局**

### 暫停與選單

- 按 **ESC** 或「⏸ 暫停」按鈕暫停（有哨音），再次操作繼續
- 按 **Q** 或左上角「←」按鈕可隨時返回主選單（比分歸零）

## 部署

### Railway（推薦）

```bash
# 安裝 Railway CLI
npm install -g @railway/cli

# 登入（使用 GitHub 帳號）
railway login

# 部署
railway up
```

或直接在 [railway.app](https://railway.app) 連結 GitHub repo，每次 `git push` 自動重新部署。

### 本機對外分享（ngrok）

```bash
# 終端機 1：啟動 Flask
python app.py

# 終端機 2：建立公開隧道
ngrok http 5000
```

將 ngrok 產生的 `https://xxx.ngrok-free.app` 網址傳給對方即可。
